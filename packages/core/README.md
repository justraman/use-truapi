# @use-truapi/core

**Framework-agnostic runtime for use-truapi.**

Host detection, chain clients, signers, transactions, chat, statements,
payments and cloud storage for Polkadot apps — the controller layer that
[`@use-truapi/react`](https://www.npmjs.com/package/@use-truapi/react) and
[`@use-truapi/vue`](https://www.npmjs.com/package/@use-truapi/vue) bind to.

Most apps should install one of those instead; reach for `@use-truapi/core`
directly only to bind another framework or to use the controllers without
one.

> **Status:** this is an experiment, not an official library. The effort will
> move into [@parity/product-sdk](https://github.com/paritytech/product-sdk)
> long term.
